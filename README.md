# FLIP — мобильный прототип

## Что уже есть
- мобильное управление;
- приятная физика платформера;
- прыжок;
- переворот гравитации;
- красивый многослойный лесной фон;
- частицы и освещение;
- кристаллы;
- враги;
- финишный портал;
- простая процедурная музыка и звуки;
- таймер и повторное прохождение.

## Как запустить
Откройте `index.html` в браузере.

Для телефона удобнее:
1. Распаковать архив.
2. Загрузить папку на любой простой статический хостинг.
3. Открыть ссылку на телефоне.

Управление на компьютере:
- A/D или стрелки — движение
- Space / W / стрелка вверх — прыжок
- F или Shift — FLIP

Это первая играбельная сборка, а не финальная игра.
